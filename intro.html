Welcome to CilioGenics

CilioGenics is a simple app and ciliary gene database providing a list of genes which we've predict to function in ciliogenesis or in related pathways. For the source code of the website: https://github.com/mustafapir/covid19-tracker

 By using this tool:

  • Any gene can be searched in the database to see if it is predicted to be ciliary,
  • Explore CilioGenics' predicted ciliary gene list
